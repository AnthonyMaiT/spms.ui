Student Point Help

This is where you can find help with student points!

Student Points are used to keep track if a student's attendance in an event. One event is one point for a student and points are resetted every quarter

Creating a Point

You can add a student point in the "Add Point" Tab on the navigation toolbar

  1. Fill out the username field and press next
  2. If the username is valid you will see their details. If not valid, go back and fill in the username correctly. Press next when you see their details
  3. At this point, you will use a selector to select an ongoing event. If there are no ongoing events, there will be an error which won't allow you to go next. When you have selected an event, press next.
  4. You will then either see an error message or a success message. You can either go back if there was an error or reset the form.

Another way to add a point is doing it the custom way in the "Student Points Tab" with the top right button

  1. Press the magnifying glass where the User field is to select a user. Then select a user
  2. Press the magnifying glass where the Event Time field is to select an event time. Then select an event time
  3. Press Create. You may see an error message if user already in event.
 Editing a point

You can edit a student point in "Student Point Records" tab and pressing the pencil icon on the far right of the datatable if there is a record you want to edit.

  1. Press the magnifying glass where the User field is to select a user. Then select a user to change
  2. Press the magnifying glass where the Event Time field is to select an event time. Then select an event time to change for the point
  3. Press Edit. You may see an error message if user already in event.
 Deleting a point

You can delete a student point in "Student Point Records" tab

  1. Press the trash can icon on the far right of the datatable if there is the record you want to delete
  2. Press the confirm button to confirm this action
 Exporting a point

You can export a student point in "Student Point Records" tab

  1. Select a quarter range you want to export points of with the magnifying glass under the quarter filter field
  2. Press the pink "Export Points to Excel" button to get a excel file to analyze or view data
Winners Help

This is where you can find help with student winners!

Winners are students who are either chosen at random for each grade or accumulated the most amount of points for the quarter range. They are also given a prize!

Creating winners

You can create winners in the "Winners" tab

  1. Press the "Create Winners" button
  2. Press the magnifying glass to select a quarter range to create winners from
  3. Press Create. If no winners are created, there will be an error.
 Editing a winner's prize

You can edit the prize of winners in the "Winners" tab.

  1. Press the pencil icon on the far right of the datatable
  2. Press the magnifying glass to select a prize you want the student to have
  3. Press Edit
 Deleting winners

You can delete winners in the "Winners" tab

  1. Press the red trashcan button next to the record you want to delete
  2. Press the confirm button to confirm your delete
Prizes Help

This is where you can find help with prizes!

The prizes tab is to show what prizes are avaliable for winners

Creating a prize

You can create prize in the "Prizes" tab

  1. Press the "Create New Prize" button
  2. Type in a name of the prize you are giving out. Make sure it doesn't match any of the other prizes' names
  3. Select the level for the prize. Level 1- 0 to 4 points, Level 2 - 5 to 14 points, Level 3 - 15+ points
  4. Press Create. If prize name matches another prize, it will return an error.
 Editing a prize

Quarters Help

This is where you can find help with quarters!

Student points, event times and winners are dependent on the quarter they are in. Set a quarter to add event times and points.

Creating a quarter

You can create an event in the "Quarters" tab

  1. Press the "Create New Quarter Range" button
  2. Select the name of the quarter you are creating
  3. Select the start and end date of the quarter. Make sure the start date is less than the end date. Also make sure they don't conflict with other quarters
  4. Press Create. You would see an error message if quarter ranges conflict with each other
 Editing a quarter

You can edit a quarter in the "Quarters" tab.

Users Help

This is where you can find help with users!

Users of the app that can log in. Have student, staff, and admin roles.

Creating a user

You can create a user in the "Users" tab

  1. Press the "Create New User" button
  2. Type in the username the user would log in with in the username field
  3. Type in the password the user would use for logging in the password field
  4. Type in the first and last name of the user in the first and last name fields
  5. Select the role type of the user. If the user is a student, select the grade level as well
  6. Press Create. You would see an error message if other usernames conflict with each other
 Editing a user

You can edit a user in the "Users" tab.

  1. Press the pencil icon on the far right of the datatable
  2. Type in the new username the user would log in with in the username field
  3. Type in the first and last name of the user in the first and last name fields
  4. Select the role type of the user. If the user is a student, select the grade level as well
  5. Press Create. You would see an error message if other usernames conflict with each other
 Rest a user's password

You can reset a user's password in the "Users" tab

  1. Press the black key button next to the record you want to delete
  2. Fill in the new password and confirm the new password
  3. Press submit
 Deleting a user

You can delete a user in the "Users" tab

  1. Press the red trashcan button next to the record you want to delete
  2. Press the confirm button to confirm your delete
